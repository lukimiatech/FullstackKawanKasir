PROMPT UNTUK CODEX - SISTEM POS KAWAN KASIR

Saya ingin membuat aplikasi Point of Sale / POS bernama Kawan Kasir.

Aplikasi ini terdiri dari:

1. Backend API
   - Gunakan Spring Boot
   - Database lokal menggunakan MySQL
   - Sistem bersifat online-only, tidak perlu mode offline
   - Backend harus menyediakan REST API lengkap untuk web admin dan Android kasir

2. Web Admin
   - Web hanya untuk level admin, owner, manager, finance, inventory, dan super admin
   - Web digunakan untuk pengelolaan sistem POS, bukan untuk kasir utama
   - Kasir utama digunakan melalui aplikasi Android

3. Android App
   - Gunakan Kotlin
   - Android digunakan oleh kasir/staff toko untuk transaksi POS
   - Android login ke backend menggunakan API
   - Android tidak perlu offline mode

Nama aplikasi: Kawan Kasir

Gunakan logo Kawan Kasir sebagai dasar tema UI. Warna utama mengikuti logo:
- Primary Blue: #039CD5
- Dark Blue: #02315E
- Secondary Blue: #075D92
- Orange: #FC8200
- Yellow: #FDBD0D
- Green: #078546
- Light Green: #6ABD2D
- Background Light: #F8F9F2
- White: #FFFFFF

Tema aplikasi harus terlihat:
- Cerah
- Bersih
- Modern
- Friendly
- Profesional
- Cocok dengan logo Kawan Kasir
- Banyak menggunakan kombinasi biru, orange, kuning, hijau, dan putih
